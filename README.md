# bus lane checker
check [singapore's bus lane timings](https://www.lta.gov.sg/content/ltagov/en/who_we_are/our_work/public_transport_system/bus/bus_priority_schemes.html) at a glance for a faster journey. Works as a PWA for your iPhone.
